Top Interview Questions

Q1: Explain the difference between stack and heap memory.

Answer: Stack is faster, for static memory. Heap is dynamic memory used for variables allocated with `new`.

Q2: What is the Rule of Three/Five?

Answer: If a class needs a destructor, copy/move constructor, or copy/move assignment operator, it likely needs all.
