Basic C++ Questions

Q1: What is the difference between a pointer and a reference?

Answer: A pointer can be reassigned and can be null, while a reference must be initialized and cannot be null.

Q2: What is the use of the 'const' keyword in C++?

Answer: 'const' is used to declare immutable variables, protect arguments, return values, or member functions from modification.
