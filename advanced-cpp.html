Advanced C++ Topics

Q1: What is a smart pointer?

Answer: Smart pointers manage memory automatically using RAII. Examples: `unique_ptr`, `shared_ptr`, `weak_ptr`.

Q2: What is move semantics?

Answer: Move semantics allows resources to be transferred from one object to another, improving performance.
