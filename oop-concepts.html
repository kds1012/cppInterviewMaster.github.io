Object-Oriented Programming (OOP)

Key concepts and interview questions on OOP in C++

What are the 4 pillars of OOP?

Answer: Encapsulation, Inheritance, Polymorphism, Abstraction.

What is function overloading vs overriding?

Answer: Overloading means same function name with different parameters. Overriding means redefining base class method in derived class.

Example: Basic Class

class MyClass {
private:
  int x;
public:
  MyClass(int val) : x(val) {}
  void display() const { std::cout << x << std::endl; }
  ~MyClass() {}
};
